# ctrl-hack-del-2024

How to run this project locally

```
cd backend/
pipenv install
pipenv shell
python src/api_endpoints.py

cd ../frontend/
npm install -f
npm run dev
```

Then, open `http://localhost:3000`.

Running our Hello World side quest:

```
cd hello-world
pipenv install
pipenv shell
python hello_world_dvd.py
```
